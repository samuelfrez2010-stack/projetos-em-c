# Calculadora de IMC em C

Este é um dos meus primeiros projetos em C, desenvolvido como parte do meu plano de estudos.

## Sobre o Projeto
O programa calcula o Índice de Massa Corporal (IMC) do utilizador, solicitando o peso e a altura, e classifica o resultado de acordo com as tabelas da OMS.

## Aprendizado
Durante o desenvolvimento, enfrentei o desafio do **Buffer de Entrada** do C:
- Aprendi que o `scanf` pode deixar "lixo" na memória se o utilizador usar vírgula em vez de ponto.
- Foquei em criar instruções claras de UX (Experiência do Utilizador) para garantir que os dados sejam inseridos corretamente.
- Pratiquei o uso de `float` para garantir precisão nos cálculos decimais.

## Como Executar
Para rodar este programa, basta copiar o código do arquivo `main.c` e executar em qualquer compilador de C (como o VS Code, Dev-C++ ou até compiladores online).
