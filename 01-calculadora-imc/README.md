# Calculadora de IMC em C

Este é um dos meus primeiros projetos em C, desenvolvido como parte do meu plano de estudos.

## Sobre o Projeto
O programa calcula o Índice de Massa Corporal (IMC) do usuário, solicitando o peso e a altura, e classifica o resultado de acordo com as tabelas da OMS.

## Aprendizado
- **Entrada de dados:** Uso do `scanf` para capturar peso e altura.
- **Lógica de Decisão:** Estruturas `if`, `else if` e `else` para classificar o IMC.
- **Tratamento de Números:** Uso de `float` para lidar com casas decimais e formatação de saída com `%.2f`.
- **UX (Experiência do Usuário):** Instruções no terminal para orientar o uso do ponto decimal e evitar erros de leitura.

## Como Executar
Para rodar este programa, basta copiar o código do arquivo `main.c` e executar em qualquer compilador de C (como o VS Code, Dev-C++ ou até compiladores online).
